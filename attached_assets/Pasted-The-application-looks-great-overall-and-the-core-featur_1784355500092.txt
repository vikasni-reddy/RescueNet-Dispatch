1. Fix "How it Works" Navigation

The navigation bar contains a "How it works" link, but clicking it does nothing.

Either:

Create a proper How It Works section/page, or
Smooth scroll to the existing section if it's already present.

The section should briefly explain the workflow:

User reports an emergency
AI analyzes the report
AI extracts structured information
Priority score is generated
Incident appears on the dashboard and live map
AI recommends the nearest emergency resources
Operator dispatches a resource
Incident status is tracked until resolved
